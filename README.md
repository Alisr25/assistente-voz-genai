#  Assistente de Voz com IA Generativa

Projeto desenvolvido durante o bootcamp **GenAI & Dados** da DIO.

A aplicação cria um fluxo completo de interação por voz utilizando Inteligência Artificial:

1.  Captura de áudio do usuário
2.  Transcrição do áudio utilizando **Whisper**
3.  Processamento da pergunta com **modelo de linguagem**
4.  Conversão da resposta em áudio com **gTTS**

Assim, o usuário pode fazer perguntas por voz e receber respostas também em áudio.

---

#  Tecnologias utilizadas

* Python
* Google Colab
* Whisper (Speech-to-Text)
* OpenAI API
* gTTS (Text-to-Speech)
* JavaScript para captura de áudio

---

#  Funcionamento do projeto

Fluxo da aplicação:

```
Usuário fala 🎤
      ↓
Captura de áudio (JavaScript)
      ↓
Whisper transcreve áudio
      ↓
Modelo de linguagem gera resposta
      ↓
gTTS converte resposta em áudio
      ↓
Usuário escuta a resposta 🔊
```

---

#  Aplicação no dia a dia

O assistente pode ser utilizado para:

* Tirar dúvidas rápidas sobre programação
